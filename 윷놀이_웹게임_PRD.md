## 2. 배경 및 목적 / Latar Belakang & Tujuan

장평중학교와 인도네시아 학교 간 국제교류 수업에서 사용할 전통 윷놀이 게임을 전자칠판 웹앱으로 구현한다.
**전통 윷놀이판 구조를 그대로 사용**하되 말이 미션 칸에 도착하면 사회정서교육(SEL) 미션이 팝업으로 노출되고, 교사가 미션 성공/실패를 판정해 점수를 부여한다.

**핵심 가치: 경쟁이 아닌 협력. 완주보다 과정.**

## 4. 윷놀이판 구조 / Struktur Papan Yut

### 4-1. 전통 윷놀이판 기준

첨부 이미지(Image 1)의 전통 윷놀이판을 기준으로 구현한다.

```
구조 개요:
- 외곽 경로: 정사각형 테두리, 각 변마다 5칸(모서리 포함)
- 총 칸 수: 외곽 20칸 + 대각선 교차점 9칸(센터 1 + 대각 중간 4 + 모서리 교차 4) = 29칸
- 출발/도착: 오른쪽 하단 모서리 (출발)
- 4개의 모서리 칸: 특수 분기점 (대각선 이동 선택 가능)
- 중앙 칸: 대각선 교차점

윷판 칸 번호 정의 (전통 배치):
  출발(0) → 1 → 2 → 3 → 4 → 우하(5, 모서리)
  → 6 → 7 → 8 → 9 → 우상(10, 모서리)
  → 11 → 12 → 13 → 14 → 좌상(15, 모서리)
  → 16 → 17 → 18 → 19 → 좌하(20, 모서리)
  → 21 → 22 → 23 → 24 → 출발(날밭, 0)

대각선 경로:
  우하(5) → 대각중간A(26) → 중앙(25) → 대각중간B(27) → 좌상(15)
  우상(10) → 대각중간C(28) → 중앙(25) → 대각중간D(29) → 좌하(20)
```

### 4-2. 말길 4가지 (Image 2 기준)

전통 윷놀이의 말길은 정확히 4가지이며, 출발 위치와 윷 결과에 따라 자동 경로가 결정된다.

| 말길 | 별칭 | 경로 설명 | 총 이동 칸 수 |
|------|------|-----------|--------------|
| **1번 (동지)** | 가장 빠른 길 | 출발 → 우하 모서리(5)에서 대각선 → 중앙(25) → 대각선 → 좌상(15) → 나머지 외곽 → 날밭 | 약 16칸 |
| **2번 (춘분)** | 중간 길 A | 출발 → 우하 모서리(5)에서 대각선 → 중앙(25) → 대각선 → 좌하(20) → 나머지 외곽 → 날밭 | 약 20칸 |
| **3번 (추분)** | 중간 길 B | 출발 → 외곽 → 우상 모서리(10)에서 대각선 → 중앙(25) → 대각선 → 좌상(15) → 나머지 외곽 → 날밭 | 약 20칸 |
| **4번 (하지)** | 가장 느린 길 | 출발 → 외곽 전체 한 바퀴 → 날밭 (대각선 미사용) | 약 25칸 |

**경로 선택 규칙:**
- 말이 모서리 칸(5, 10, 15, 20)에 정확히 도달했을 때, 플레이어가 대각선 경로 또는 외곽 경로 중 선택한다.
- 모서리를 지나쳐 버린 경우(칸 수 초과) 선택 불가, 외곽 경로만 사용.
- 중앙(25)에서는 두 방향 중 선택 가능.

## 5. 게임 규칙 / Aturan Permainan

### 5-1. 윷 결과 및 이동 칸 수

| 결과 | 이동 | 추가 기회 |
|------|------|-----------|
| 백도 (뒤로) | -1칸 후퇴 | 없음 |
| 도 | 1칸 전진 | 없음 |
| 개 | 2칸 전진 | 없음 |
| 걸 | 3칸 전진 | 없음 |
| 윷 | 4칸 전진 | **한 번 더 던지기** |
| 모 | 5칸 전진 | **한 번 더 던지기** |

**백도 특별 규칙:**
- 출발 전 말(아직 보드에 나오지 않은 말)에 백도가 나오면 → 말을 내보내지 않고 차례 종료
- 이미 보드에 있는 말에 백도 적용 시 → 1칸 후퇴
- 백도로 날밭 진입 불가 (날밭 1칸 앞에 있어도 후퇴만 함)

### 5-2. 추가 던지기 규칙 (한 번 더 / Lempar Lagi)

다음 경우 해당 팀은 윷을 **한 번 더** 던질 기회를 얻는다:

1. **윷(4칸)이 나온 경우**
2. **모(5칸)이 나온 경우**
3. **상대방 말을 잡은 경우**

**중요 예외:**
- 윷 또는 모를 이용해 상대 말을 잡은 경우 → 추가 던지기는 **1번만** 허용 (중복 적용 불가)
  - 예: 윷으로 잡음 → 윷로 인한 추가 던지기 1회 (잡기로 인한 추가 던지기는 중복 불가)
- 한 턴에 여러 번 추가 던지기가 발생한 경우, 모두 모아서 순서를 자유롭게 바꾸어 사용 가능

### 5-3. 잡기 규칙 (잡기 / Menangkap)

말이 이동 후 도착한 칸에 다른 말이 있을 경우:

**① 같은 팀 말이 있을 때 → 업기 (Tumpuk)**
- 자기 팀 말 위에 포개어 함께 이동 (업기)
- 업은 말은 이후 모든 이동에서 **함께 움직임**
- 업은 말은 잡히기 전까지 **분리 불가**
- 업힌 상태에서 잡힐 경우 → 업은 말 **전부** 출발 지점으로 되돌아감

**② 상대 팀 말이 있을 때 → 잡기 (Tangkap)**
- 상대 말을 잡아 출발 위치로 돌려보냄
- 상대가 여러 말을 업고 있었다면 → 업은 말 **전부** 출발로 되돌아감
- 잡은 팀은 추가 던지기 1회 획득 (5-2 규칙 참고)

### 5-4. 말 수 설정 (말 개수 / Jumlah Bidak)

- 게임 시작 전 팀당 말 수를 **1~4개** 중 선택 (기본값: **1개**, 이번 수업 기준)
- 모든 팀이 같은 수의 말을 사용
- 모든 말이 날밭에 도달한 팀이 완주

### 5-5. 날밭(완주) 규칙

- 말이 출발 칸(날밭)을 통과하거나 정확히 도달하면 완주
- 완주 칸을 초과한 이동은 허용 (넘어서 도달해도 완주 처리)
- 완주 시 +20점 보너스

## 6. 기능 요구사항 / Persyaratan Fungsional

### 6-1. 게임 설정 화면

- [ ] 팀 수 입력 (기본값: 9, 최소 2, 최대 12)
- [ ] 팀별 이름 입력 (기본값: 팀 1~9 / Tim 1~9)
- [ ] 팀당 말 수 선택 (1~4개, 기본: 1개)
- [ ] 미션 성공 점수 설정 (기본: +10점)
- [ ] 미션 실패 점수 설정 (기본: -5점)
- [ ] 완주 보너스 점수 설정 (기본: +20점)
- [ ] 게임 시작 버튼
- [ ] localStorage에 설정값 저장

### 6-2. 게임 보드 (전통 윷놀이판)

- [ ] 전통 윷놀이판 SVG 렌더링 (정사각형, 대각선 포함)
- [ ] 칸 종류:
  - **일반 칸**: 이동만 함 (작은 원)
  - **모서리 칸 (4개)**: 대각선 분기 선택 가능 (큰 원, 강조 표시)
  - **중앙 칸**: 대각선 교차점 (가장 큰 원)
  - **출발/날밭 칸**: 우하단 (출발 텍스트 표기)
  - **미션 칸 (? 아이콘)**: 4~5곳 배치, 도착 시 미션 팝업
  - **행운 칸 (★ 아이콘)**: 1~2곳 배치, 도착 시 추가 점수
- [ ] 팀별 말을 보드 위에 시각적으로 표시 (팀 색상 구분)
- [ ] 업은 말은 겹쳐진 원 형태로 표시 (숫자로 몇 개 업었는지 표기)
- [ ] 말 경로 선택 UI: 모서리 도달 시 "대각선으로 가기 / Jalur Diagonal" vs "외곽으로 가기 / Jalur Luar" 선택 버튼 표시

### 6-3. 윷 결과 입력 / Input Hasil Yut

> **변경 사항 (v2 개정):** 컴퓨터가 결과를 시뮬레이션하지 않는다.
> 실제 학생 4명이 실제 윷을 던진 후, 나온 결과를 진행 교사가 화면에서 버튼으로 선택 입력한다.

- [ ] "결과 입력 / Input Hasil" 버튼 클릭 시 결과 선택 다이얼로그 표시
- [ ] 다이얼로그에 6개 결과 버튼을 크게 표시:
  - **백도** / Mundur Satu (−1칸)
  - **도** / Do (1칸)
  - **개** / Gae (2칸)
  - **걸** / Geol (3칸)
  - **윷** / Yut (4칸 + 한 번 더)
  - **모** / Mo (5칸 + 한 번 더)
- [ ] 버튼 클릭 즉시 결과 등록, 다이얼로그 닫힘
- [ ] 윷/모 선택 시 "한 번 더! (Lempar Lagi)" 알림 표시
- [ ] 등록된 결과는 대기 목록에 칩으로 표시, 순서 변경 가능

### 6-4. 말 이동 / Gerakkan Bidak

- [ ] 윷 결과 선택 → 적용할 말 선택 → 이동 실행
- [ ] 모서리 도달 시 경로 선택 다이얼로그
- [ ] 칸 단위 이동 애니메이션
- [ ] 이동 후 이벤트 자동 처리:
  - 같은 팀 말 만남 → 업기 확인 다이얼로그
  - 상대 팀 말 만남 → 잡기 처리 + 추가 던지기 획득
  - 미션 칸 도달 → 미션 팝업
  - 행운 칸 도달 → 점수 보너스
  - 날밭 도달 → 완주 처리 + 점수 보너스

### 6-5. 미션 팝업 / Popup Misi

- [ ] 화면 중앙 오버레이 팝업
- [ ] 구성 요소:
  - SEL 영역 태그 (자기 / 대인관계 / 공동체 / 마음건강)
  - 미션 제목 (한국어 + 인도네시아어)
  - 미션 내용 (한국어 + 인도네시아어)
  - 1분 카운트다운 타이머 (프로그레스바)

### 7-2. 언어 병기 원칙

- 타이틀, 버튼, 메뉴, 팝업, 미션 내용 모두 **한국어 + 인도네시아어** 병기
- 한국어 위(크게, 진하게), 인도네시아어 아래(작게, 이탤릭, 연한 색)
- 영어 미사용

### 7-4. 전자칠판 최적화

- 최소 해상도: 1280×720, Full HD 1920×1080 기준 설계
- 버튼 최소 크기: 48×48px (터치 가능)
- 보드 칸 최소 크기: 40px 직경
- 고대비 텍스트 (어두운 갈색 배경에 진한 텍스트 또는 반대)

### 7-5. 애니메이션

- 윷 결과 입력: 애니메이션 없음 (실제 윷 던지기 후 결과 버튼 선택)
- 말 이동: 칸 단위 슬라이드 (0.3초/칸)
- 잡기 이벤트: 말이 터지는 효과 후 출발로 이동
- 업기 이벤트: 말이 포개지는 애니메이션
- 미션 팝업: 페이드인 + 스케일업 (0.3초)
- 점수 변경: 숫자 카운트 애니메이션 (0.5초)
- 미션 타이머: 1분 프로그레스바 (60초 → 0초, 30초 미만 주황, 10초 미만 빨강)
